## Basic Shortcuts

Shortcuts I find useful

| Shortcut           | Desc                  |
| ------------------ | --------------------- |
| `space`            | Pan                   |
| `z` + `click`      | Zoom In               |
| `z`+ `⌥` + `click` | Zoom Out              |
| `⇧` + `1`          | Zoom Out All          |
| `⇧` + `2`          | Zoom Out On Selection |
| `f`                | Frame Tool            |
| `v`                | Move Tool             |
| `r`                | Rectangle Tool        |
| `⌘` + `⌥` + `k`    | Create Components     |
| `⌘` + `g`          | Group                 |
| `⌘` + `⌥`+ `g`     | Frame Group           |
| `⌘` + `⇧` + `g`    | Ungroup               |
| `⌘` + `⌥` + `c`    | Copy Styles           |
| `⌘` + `⌥` + `v`    | Paste Styles          |
