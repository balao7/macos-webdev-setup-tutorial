## Basic Shortcuts

Shortcuts I find useful

| Shortcut             | Desc              |
| -------------------- | ----------------- |
| space                | Pan               |
| cmd + space          | Zoom In           |
| cmd + space + option | Zoom Out          |
| f                    | Frame             |
| v                    | Move              |
| r                    | Rectangle         |
| cmd + option + k     | Create Components |
| cmd + g              | Group             |
| cmd + shift + g      | Ungroup           |
| cmd + alt + c        | Copy Styles       |
| cmd + alt + v        | Paste Styles      |
