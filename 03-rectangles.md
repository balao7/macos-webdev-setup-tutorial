## Basic Shortcut

Shortcuts I find useful

| Shortcut             | Screen      |
| -------------------- | ----------- |
| `⌃` + `⌥` + `←`      | left half   |
| `⌃` + `⌥` + `→`      | right half  |
| `⌃` + `⌥` + `↑`      | top half    |
| `⌃` + `⌥` + `↓`      | bottom half |
| `⌃` + `⌥` + `return` | fullscreen  |
| `⌃` + `⌥` + `⌫`      | reset       |
| `⌃` + `⌥` + `e`      | 2/3         |
| `⌃` + `⌥` + `d`      | 1/3         |
